# Movie-correlation-Project-using-Python
Exploring Correlations in Movie Data  Dive into the world of movies and uncover hidden relationships between budget, ratings, genres, and more! This project delves into a comprehensive dataset of 6820 movies to uncover intriguing correlations and patterns. Using Python tools like pandas, NumPy, matplotlib, and Seaborn. 

visualize and analyze aspects such as:

Financial Factors: Explore the connections between budget, gross revenue, ratings, and genres.
Critical vs. Popular Success: Investigate the relationship between critic scores and audience scores.
Genre Trends: Uncover patterns in genre popularity over time and identify which genres tend to have higher or lower ratings.
Star Power: Analyze the impact of star actors on movie performance.
Time-Based Insights: Discover trends in release dates, runtimes, and ratings across different years and decades.

Data Source: https://www.kaggle.com/datasets/danielgrijalvas/movies/data

Tools and Libraries: pandas, NumPy, matplotlib, Seaborn
